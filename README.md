# PixelAI

PixelAI is a sophisticated Image Discovery Engine built to showcase high-end UI/UX using modern React, Framer Motion, and Tailwind CSS.

## Identity-Aware Architecture

This project strictly adheres to the "Antigravity UI" design system:
1. **Glassmorphism Layering**: Depth is achieved using `backdrop-blur` and low-opacity structural layers over a dark (`slate-950`) space-themed radial gradient base.
2. **Neon Gradients & Accent Colors**: Employs deep purples and pinks for energetic micro-interactions without overwhelming the focus on content.
3. **Motion-First Philosophy**: Uses spring-based physics for entrances and hover states, giving the layout an organic, premium feel.

## Tech Stack
- Frontend: React 18 / Vite
- Styling: Tailwind CSS
- Animation: Framer Motion
- Icons: Lucide-React
- API: Axios / Unsplash API

## Running Locally

1. Setup `.env` file with Unsplash Access Key.
2. `npm install`
3. `npm run dev`
